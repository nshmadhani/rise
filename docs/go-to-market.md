# Go-to-market — Rise

**Acquisition: paid ads only.**  
Organic (Reddit/IH/etc.) is out of scope — category is saturated; easy to get overlooked.

## Funnel

Ad → https://riseto.app → waitlist email

## Measure

| Metric | Source |
|--------|--------|
| Visitors | Vercel Analytics |
| Waitlist emails | Gist / export API |
| Conversion % | emails ÷ visitors |
| CPA | ad spend ÷ emails |

## Cadence

1. Ship one creative + one landing headline  
2. Spend a fixed small budget  
3. Read CPA + conversion  
4. Change **one** variable (creative *or* headline)  
5. Repeat
